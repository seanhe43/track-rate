# soundtrack-r

A full-stack music rating and discovery application built with React, Node.js, and the Spotify API ecosystem. Users can search for albums, listen directly in-app through Spotify playback integration, rate music, and maintain personal listening notes and collections.

## Frontend
JavaScript + React
Component-based UI architecture
Dynamic album card rendering
Responsive grid layouts
State-driven filtering and sorting systems

## Backend
Node.js backend services
Spotify API integration
OAuth authentication flow handling
API token management for search and playback


### Spotify Integration
Spotify Web API integration
Spotify Player SDK integration
Spotify OAuth authentication
In-app music playback support

## User Features
Album search functionality
Music playback within the app
Album rating system
Personal listening notes
User music library tracking
Sorting and filtering of rated albums
